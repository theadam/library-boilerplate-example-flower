# library-boilerplate-example-flower

A boilerplate to be used with [pollinate](https://github.com/everysquare/pollinate) to create an example inside a library.

# Variable Stuff

```
cat > example.json << CONFIG
{
  "name": "example-name",
  "context": {
    "baseName": "project",
    "githubUser": "theadam"
  }
}
CONFIG
```
